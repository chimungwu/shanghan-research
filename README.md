# 傷寒論方劑性味分析（趙開美翻刻宋本）

以《神農本草經》性味數據為基礎，結合藥物用量權重，分析趙開美翻刻宋本《傷寒論》116 首方劑的五味四氣偏性。

## 功能

- 116 首方劑的五味（酸苦甘辛鹹）與四氣（寒熱溫涼平）加權分析
- 五角雷達圖呈現五味分佈，支援方劑對比
- 寒熱指數量化與偏性判定
- 305 條宋本原文條文對照
- 依六經分類、偏性篩選與搜尋
- 散佈圖視覺化總覽

## 典籍版本

- **方劑來源**：趙開美翻刻宋本《傷寒論》（明萬曆二十七年刊本）
- **性味來源**：《神農本草經》
