# 傷寒論方劑性味分析

以《神農本草經》五味四氣數據為基礎，結合藥物用量加權計算，對趙開美翻刻宋本《傷寒論》112 首方劑進行性味結構量化分析。

## 檔案結構

```
shanghan-app/
├── index.html          # 主頁面（可直接開啟，載入 data.js + app.js）
├── app.jsx             # React 原始碼（供後續重構 / App 化）
├── data.js             # 瀏覽器版資料入口（供網站直接載入）
├── app.js              # 瀏覽器實際執行的編譯版
├── core.browser.js     # 瀏覽器可直接使用的核心邏輯入口
├── data/
│   ├── formulas.json   # 112 首方劑基本資料（性味分析、用量、組成原文）
│   ├── herbs.json      # 86 味藥物的《神農本草經》原文與按語
│   ├── compare-groups.json # 預設比較組
│   └── clauses.json    # 273 條方劑相關條文（含宋桂康三版本異文）
├── src/
│   └── core/           # 未來離線 App 可共用的核心模組
├── APP_MIGRATION.md    # App 化遷移說明
├── README.md
└── LICENSE
```

## 部署方式

### 本地使用 / 靜態部署

目前網站採用「頁面殼 + 資料層 + 編譯後主程式」結構：

- `index.html`：可直接用瀏覽器開啟
- `data.js`：提供網站直接使用的資料
- `core.browser.js`：把共用核心邏輯掛到 `window.AppCore`
- `app.js`：瀏覽器執行版本
- `app.jsx`：保留作為後續重構與 App 化的原始碼

若要用本地伺服器開啟，也可以：

```bash
# Python
python3 -m http.server 8000

# Node.js
npx serve .
```

### iOS App 開發

資料層已分離為獨立 JSON 檔案，可直接作為 SwiftUI / React Native 的資料來源：

- `data/formulas.json` → 方劑列表、性味分析結果
- `data/herbs.json` → 藥物本草經原文（key: 藥名, value: 原文+按語）
- `data/clauses.json` → 條文對照（key: 方劑名, value: 條文陣列含異文）

## 資料格式

### formulas.json

```json
{
  "n": "桂枝湯",          // 方名
  "c": "太陽病",          // 六經分類
  "tw": 14.6,            // 總用量（兩）
  "fp": {"辛":41.1,"甘":58.9,"苦":20.5},  // 五味百分比
  "pf": "甘",            // 主味
  "fr": ["甘","辛","苦"], // 味排序
  "ns": 0.89,            // 寒熱指數
  "nt": "偏溫",          // 偏性判定
  "ng": {"寒涼":0,"溫熱":41.1,"平":38.4},  // 四氣構成
  "hd": [                // 藥物組成
    {"n":"桂枝","w":3,"f":["辛","甘"],"t":"溫"},
    ...
  ],
  "hl": "桂枝（三兩，去皮）...",  // 組成原文
  "cm": "右五味，㕮咀三味..."     // 煎服法原文
}
```

### herbs.json

```json
{
  "桂枝": "牡桂　味辛溫。主治上氣咳逆...\\n【按】桂枝為牡桂之嫩枝...",
  "甘草": "甘草　味甘平。主治五臟六府...\\n【按】炙甘草：經蜜炙後...",
  ...
}
```

### clauses.json

```json
{
  "桂枝湯": [
    {
      "num": 12,
      "t": "太陽中風，脈陽浮而陰弱...",
      "d": [
        {"v":"桂","t":"太陽中風，陽浮而陰弱..."},
        {"v":"康","t":"大陽中風，陽浮而陰弱..."}
      ]
    },
    ...
  ]
}
```

## 典籍版本

- **方劑來源**：趙開美翻刻宋本《傷寒論》（明萬曆二十七年刊本）
- **性味來源**：《神農本草經》（清·孫星衍輯本）
- **比較版本**：桂林古本《傷寒雜病論》、康平本《傷寒論》
