# 傷寒論方劑性味分析

以《神農本草經》五味四氣數據為基礎，結合藥物用量加權計算，對趙開美翻刻宋本《傷寒論》112 首方劑進行性味結構量化分析。

## 檔案結構

```
shanghan-app/
├── index.html          # 主頁面（載入外部 app.jsx）
├── standalone.html     # 獨立版（單檔部署用）
├── app.jsx             # React 主程式（所有 UI 組件）
├── data/
│   ├── formulas.json   # 112 首方劑基本資料（性味分析、用量、組成原文）
│   ├── herbs.json      # 86 味藥物的《神農本草經》原文與按語
│   └── clauses.json    # 273 條方劑相關條文（含宋桂康三版本異文）
├── README.md
└── LICENSE
```

## 部署方式

### GitHub Pages（推薦）

使用 `standalone.html`，重新命名為 `index.html` 後上傳即可。所有資料內嵌於單一檔案中。

### 本地開發 / 進階部署

使用 `index.html` + `app.jsx` + `data/` 的組合。需要 HTTP 伺服器（Babel 的 `src` 載入需要 HTTP）：

```bash
# Python
python3 -m http.server 8000

# Node.js
npx serve .
```

### iOS App 開發

資料層已分離為獨立 JSON 檔案，可直接作為 Swift/React Native 的資料來源：

- `data/formulas.json` → 方劑列表、性味分析結果
- `data/herbs.json` → 藥物本草經原文（key: 藥名, value: 原文+按語）
- `data/clauses.json` → 條文對照（key: 方劑名, value: 條文陣列含異文）

## 資料格式

### formulas.json

```json
{
  "n": "桂枝湯",          // 方名
  "c": "太陽病",          // 六經分類
  "tw": 14.6,            // 總用量（兩）
  "fp": {"辛":41.1,"甘":58.9,"苦":20.5},  // 五味百分比
  "pf": "甘",            // 主味
  "fr": ["甘","辛","苦"], // 味排序
  "ns": 0.89,            // 寒熱指數
  "nt": "偏溫",          // 偏性判定
  "ng": {"寒涼":0,"溫熱":41.1,"平":38.4},  // 四氣構成
  "hd": [                // 藥物組成
    {"n":"桂枝","w":3,"f":["辛","甘"],"t":"溫"},
    ...
  ],
  "hl": "桂枝（三兩，去皮）...",  // 組成原文
  "cm": "右五味，㕮咀三味..."     // 煎服法原文
}
```

### herbs.json

```json
{
  "桂枝": "牡桂　味辛溫。主治上氣咳逆...\\n【按】桂枝為牡桂之嫩枝...",
  "甘草": "甘草　味甘平。主治五臟六府...\\n【按】炙甘草：經蜜炙後...",
  ...
}
```

### clauses.json

```json
{
  "桂枝湯": [
    {
      "num": 12,
      "t": "太陽中風，脈陽浮而陰弱...",
      "d": [
        {"v":"桂","t":"太陽中風，陽浮而陰弱..."},
        {"v":"康","t":"大陽中風，陽浮而陰弱..."}
      ]
    },
    ...
  ]
}
```

## 典籍版本

- **方劑來源**：趙開美翻刻宋本《傷寒論》（明萬曆二十七年刊本）
- **性味來源**：《神農本草經》（清·孫星衍輯本）
- **比較版本**：桂林古本《傷寒雜病論》、康平本《傷寒論》
