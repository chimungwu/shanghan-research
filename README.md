# 傷寒論方劑性味分析研究

本網站是一個以《傷寒論》方劑為核心的數位研究工具。  
研究目標不是取代經典閱讀與臨床辨證，而是提供一個可重複、可比較、可視覺化的分析框架，協助使用者從「性味結構」重新理解仲景組方邏輯。

## 一、研究方式

### 1. 研究底本與資料範圍
- 方劑底本：趙開美翻刻宋本《傷寒論》。
- 以宋本方劑為主體建立資料庫。
- 全書 113 首中，收錄可量化分析之 112 首（燒裩散不納入量化；禹餘糧丸原文方闕未收錄）。

### 2. 性味資料來源
- 主要依據《神農本草經》記載之五味四氣。
- 對於本草經未獨立收錄或名稱有異者，建立藥名對應與必要補充規則。

### 3. 量化方法
- 將每味藥之性味屬性與方中用量結合，計算方劑五味分佈。
- 將四氣轉為寒熱分數，依用量加權計算方劑寒熱指數。
- 結果以可視覺化方式呈現，便於跨方比較與教學討論。

## 二、這個網站的目的

- 讓《傷寒論》方劑從「文字記憶」進一步進入「結構理解」。
- 提供中醫學習者、研究者、臨床醫師一個可操作的方劑分析平台。
- 建立一個能夠持續擴充的研究介面，連結經典文本、藥物性味、條文異文與測驗回饋。

## 三、核心功能

- 方劑分析：查看單方五味、寒熱、藥物組成、條文內容。
- 方劑比較：多方疊合比較，檢視共同藥、獨有藥與關鍵差異。
- 方劑測驗：題型選擇、題數設定、錯題記錄與複習。
- 研究說明：公開資料來源、操作定義、研究限制與爭議處理方式。

## 四、與其他網站的不同

- 以「宋本方劑研究」為主軸，不是單純方劑查詢或背誦工具。
- 以「性味量化」作為核心方法，能做跨方劑的一致性比較。
- 同時整合條文異文對照（宋本／桂林古本／康平本）與方劑分析。
- 內建測驗與錯題機制，形成「學習－檢核－再學習」循環。
- 清楚揭露資料口徑與限制，方便學術討論與版本追蹤。

## 五、使用聲明
